# TWRP-P8LiTE

TWRP image for Huawei P8 Lite 

HOWTO: 

    cd TWRP-P8LiTE/.scripts
    ./repack.sh ../TWRP-P8LiTE
    fastboot flash recovery_ramdisk image-file.img
    
    That's all!
    
REQUIREMENTS

Huawei P8 Lite
Repack toool (in _scripts dir)
